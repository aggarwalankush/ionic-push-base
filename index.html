Ionic 2 Push Base

This can be used as base template for Ionic 2 Push apps. It's working for both iOS and Android Push Notifications.

Tutorial

  • Push Notifications in Ionic 2
  • FCM Setup for Android Notifications
  • APNS Setup for IOS Notifications

Getting Started

  • Clone this repository

  • Install Ionic, cordova and node_modules

    $ npm install -g ionic
$ sudo npm install -g cordova@6.3.1
$ npm install

  • Generate SENDER_ID using this tutorial ğŸ‘

  • Replace YOUR_SENDER_ID in config.xml and app.ts with above SENDER_ID

Android

    $ ionic platform add android
    $ ionic build android
    $ ionic run android

iOS

    $ ionic platform add ios
    $ ionic build ios
Run using XCode

Use device token printed in console for push notifications using below server code
